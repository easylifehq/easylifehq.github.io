Quick Guide

How to use this page

One row = one task

Add a single clear task per row so your planner stays easy to scan.

Keep titles short

Start with the action: call, email, return, clean, schedule, buy, submit.

Use notes for context

Put extra details in notes so the title stays clean on the dashboard.

Be honest with priority

Only mark things high priority when they actually deserve attention.
