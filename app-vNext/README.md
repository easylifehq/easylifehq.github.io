# Easy System App vNext

This is the React + Firebase shell for the next version of EasyHQ, EasyList, and EasyCalendar.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Current scope

- shared Firebase auth wiring
- authenticated app shell
- placeholder routes for EasyHQ, EasyList, EasyCalendar, and Settings
- typed Firestore collection helpers

## Notes

- this app lives alongside the current static site for now
- the migration target is EasyList first, then EasyCalendar, then EasyHQ
